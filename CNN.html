Resolucin de Digit Recognition usando una Red Neuronal Convolucional (CNN)

En esta libreta exploraremos la resolucin del problema de digit recognition a travs de un tipo concreto de red neuronal, la red neuronal convolucional.

Una red neuronal convolucional (CNN) es un modelo de aprendizaje profundo especializado en procesar imgenes. Utiliza capas de convolucin para detectar patrones, activaciones no lineales para complejidad, agrupacin para reducir dimensiones y capas densas para clasificacin. Durante el entrenamiento, ajusta sus pesos para hacer predicciones precisas en nuevas imgenes, siendo ampliamente aplicada en tareas de visin por computadora.

Creamos y compilamos el modelo

Nuestro modelo se compone de las siguientes capas:

  • Capas Convolucionales (Conv2D):

    Utilizadas para detectar patrones locales en la imagen mediante filtros de convolucin. La primera capa convolucional tiene 32 filtros de tamao (3, 3) y la segunda tiene 64 filtros. Se aplica la funcin de activacin ReLU para introducir no linealidades.

  • Capas de Agrupacin (MaxPooling2D):

    Reducen las dimensiones espaciales de los datos al seleccionar los valores mximos en regiones especficas. Se utilizan despus de las capas convolucionales para conservar la informacin relevante.

  • Capa de Aplanado (Flatten):

    Transforma la salida de las capas convolucionales y de agrupacin en un vector unidimensional. Prepara los datos para ser utilizados en capas densas (totalmente conectadas).

  • Capas Densas (Dense):

    En la primera capa densa, se utilizan 128 neuronas con la funcin de activacin ReLU. La ltima capa densa tiene 10 neuronas con la funcin de activacin softmax, apropiada para problemas de clasificacin con 10 clases.

from keras.models import Sequential
from keras.layers import Dense
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

input_shape = (28, 28, 1)

# Create a Sequential model
model = Sequential()

# Convolutional and Pooling Layers
model.add(Conv2D(32, kernel_size=(3, 3), activation='relu', input_shape=input_shape))
model.add(MaxPooling2D(pool_size=(2, 2)))
model.add(Conv2D(64, kernel_size=(3, 3), activation='relu'))
model.add(MaxPooling2D(pool_size=(2, 2)))

# Flatten Layer to transition from convolutional to dense layers
model.add(Flatten())

# Dense Layers
model.add(Dense(128, activation='relu'))
model.add(Dense(10, activation='softmax'))

model.summary()
